# Cooling Down The House
一款为**最终幻想14**装修玩家制作的傻瓜级家具装修软件

目前适用于国服版本**7.51**，<br>
仅在国服测试过使用，其他如国际服等请自行测试

本工具仅用于个人装修兴趣研究与技术交流，不包含任何破坏游戏平衡的功能。
因使用本工具导致的任何账号处罚或损失，均由使用者本人自行承担，作者不承担任何责任。理性使用。

## 使用说明
1. 下载最新版zip文件，解压到任意空文件夹
2. 打开游戏，**右键以管理员身份运行**“CoolingDownTheHouse.exe”
3. 在**旋转模式**下选中**家具/庭具**以读取或调整参数
4. 左键空地或其他家具保存位置（右键为取消变更）

## 功能简介
1. 家具坐标读取与更改
2. 保存家具坐标并粘贴予其他家具
3. +-按钮短按数值增减0.001，长按持续增减
4. 点击文本框里的数字左右拖动也可以增减数值
5. 左右拖动的数值由最上方的滑块调整
6. 勾选置顶窗口以保持此程序窗口悬浮在游戏窗口上
